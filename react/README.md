## What is React app structure

```
node_modules - js components
public - static content with index.html having '<div id="root"></div>'
src
 - components
   - App.js with "<BrowserRouter>"
 - index.js having
        ReactDOM.render(
            <BrowserRouter>
            <App />
            </BrowserRouter>,
            document.getElementById('root'));
```
